格式指定字 format specifier
===
```
#include <stdio.h>
int main(){
	printf("顯示字元 %c\n",'A');
	printf("顯示字元編碼 %d\n",'A');
	printf("顯示字元編碼 %c\n",65);
	printf("顯示十進位整數 %d\n",15);
	printf("顯示八進位整數 %o\n",15);
	printf("顯示十六進位整數 %X\n",15);
	printf("顯示十六進位整數 %x\n",15);
	printf("顯示十六進位整數 %E\n",0.001234);
	printf("顯示十六進位整數 %e\n",0.001234);	
}
```
